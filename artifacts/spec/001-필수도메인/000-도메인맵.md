## 필수 도메인 맵

| 도메인 | 핵심 역할 (Core Responsibility) | 타 도메인과의 주요 연결점 |
| --- | --- | --- |
| User | 주체(Actor) 식별 및 소속 관리 | 조직/직급 정보를 MDM에서 참조 |
| Audit | 행위 증적 및 데이터 무결성 보장 | 모든 도메인의 행위를 기록, Notification 보안 알림 유발 |
| Menu/Perm | 인가 엔진 및 기능 접근 제어 | Shift의 근무 여부, Approval의 승인 결과와 연동 |
| Shift | 시간/자원 기반의 동적 제어 | 근무 시간 외 접근 시 Menu/Perm의 권한 일시 차단 |
| Approval | 의사결정 프로세스 및 책임 소재 | 승인 완료 시 Menu/Perm 권한 프로비저닝 트리거 |
| MDM | 표준 코드, 서열, 기준 일정 관리 | User(직급), Approval(서열), Shift(달력)의 기준 제공 |
| Notification | 이벤트 전파 및 채널별 통지 | Approval(승인요청), Audit(보안경고) 등의 상태 변화 알림 |
